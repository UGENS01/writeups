# OverTheWire Bandit — Level 0

## Goal
Connect to the Bandit server using SSH

## Commands Used
ssh bandit0@bandit.labs.overthewire.org -p 2220
ls
cat readme

## Solution
Connected to the server using SSH on port 2220
with username bandit0 and password bandit0.
Used ls to list files in the directory.
Used cat to read the readme file which 
revealed the password for Level 1.

## What I Learned
- How SSH works and its basic syntax
- ls lists files in a directory
- cat reads and displays file contents
- Passwords never show on screen while typing in Linux
- cls is Windows only — Linux uses clear.
